# webgl-presentation

Presentation on [WebGL](https://en.wikipedia.org/wiki/WebGL), [Three.js](https://threejs.org/) and Shaders presented at the [Barcamp Salzburg](https://barcamp-sbg.at/). The [Slides](http://janakiev.com/webgl-presentation/) are done with the HTML presentation framework [Reveal.js](https://github.com/hakimel/reveal.js/) and are using [browserify](https://www.npmjs.com/package/browserify), [watchify](https://www.npmjs.com/package/watchify) and [glslify](https://www.npmjs.com/package/glslify).
